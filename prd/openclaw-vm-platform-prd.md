# OpenClaw VM Platform - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
OpenClaw VM Platform 是一个虚拟机租赁平台，让用户可以快速拥有自己的 OpenClaw 实例，无需复杂的技术配置。

### 1.2 核心价值
- **一键部署**：自动创建虚拟机并配置 OpenClaw
- **完全隔离**：每个用户独立的 VM，数据安全
- **灵活配置**：多种套餐和 Agent 模板可选
- **简化运维**：AI 辅助运维，降低管理成本

### 1.3 目标用户
| 用户类型 | 场景 | 付费意愿 |
|---------|------|---------|
| 个人开发者 | 日常助手、代码审查 | 中 |
| 小团队 | 协作、自动化 | 高 |
| 企业客户 | 客服、运维、数据分析 | 很高 |

## 2. 核心功能

### 2.1 VM 创建流程

```
用户选择套餐
    ↓
选择 Agent 模板
    ↓
支付订单
    ↓
自动创建 VM
    ↓
配置渠道（向导）
    ↓
开始使用
```

### 2.2 功能模块

#### 1. 用户系统
- 注册/登录（邮箱、OAuth）
- 用户中心
- 订单管理

#### 2. VM 管理
- 创建 VM（选择套餐、配置）
- VM 列表和详情
- 启动/停止/删除
- SSH 访问（高级用户）

#### 3. Agent 市场
- 预设 Agent 模板（客服、开发、分析等）
- Agent 详细介绍（灵魂、人设、能力）
- 自定义 Agent 创建（AI 辅助生成配置）

#### 4. 渠道配置
- 飞书 Bot 配置向导
- 飞书应用创建引导
- 事件订阅配置
- 权限设置向导

#### 5. 计费系统
- 套餐订阅（按月/年）
- Token 用量计费
- 账单和发票

#### 6. 监控面板
- VM 状态监控
- Token 使用统计
- Agent 活跃度
- 成本分析

## 5. 技术架构

### 5.1 虚拟化支持

| 方式 | 适用场景 | 实现难度 |
|-----|---------|---------|
| **KVM** | 自建机房、私有云 | 中 |
| **OpenStack** | 大规模部署 | 高 |
| **云厂商 API** | 公有云（阿里云等） | 低 |

### 5.2 镜像管理
- 基础镜像：预装 Node.js、Docker、OpenClaw
- 场景镜像：包含特定 Agent 模板
- 版本管理：支持 OpenClaw 版本选择

### 5.3 自动化部署
```bash
# 用户选择配置后，自动执行：
1. 调用云 API 创建 VM
2. 等待 VM 启动
3. SSH 连接
4. 安装 OpenClaw
5. 配置 Agent
6. 启动服务
7. 返回访问信息
```

### 5.4 渠道配置流程（飞书）
```
1. 引导用户创建飞书自定义应用（开放平台）
2. 用户输入 App ID 和 App Secret
3. 测试凭证有效性
4. 配置事件订阅地址
5. 配置所需权限（消息、通讯录等）
6. SSH 到 VM 写入配置
7. 重启服务
8. 返回应用配置页面链接
```
