## 1. 产品概述

### 1.1 产品定位
OpenClaw VM Platform 是一个 OpenClaw 虚拟机租赁平台，让用户可以快速拥有自己的 OpenClaw 实例，无需复杂的技术配置。

### 1.2 核心价值
- **开箱即用**：选择套餐 → 自动创建 → 立即使用
- **完全隔离**：每个用户独立的 VM，数据安全
- **灵活配置**：可调整 Agent、配置大模型
- **按量计费**：Token 消耗 + VM 规格

### 1.3 目标用户
| 用户类型 | 场景 | 付费意愿 |
|---------|------|---------|
| 个人用户 | 日常助手、日程管理 | 中 |
| 小团队 | 数据分析、报告生成 | 高 |
| 企业客户 | 财务分析、风险评估 | 很高 |

## 2. 用户流程

### 2.1 完整流程

```
1. 注册/登录
   └─ 邮箱注册，邮箱验证，无短信
   
2. 选择套餐
   └─ 选择使用场景（个人助理/数据分析/财务分析等）
   └─ 套餐包含预设 Agent + 固定 VM 规格
   
3. 调整 Agent（可选）
   └─ 可删除套餐中的 Agent
   └─ 可添加其他预设 Agent
   └─ 可自定义创建 Agent（引导式）
   
4. 配置大模型
   └─ 选择使用平台模型
   └─ 或使用自己的 API Key
   
5. 确认创建
   └─ 点击【确认】按钮
   
6. 自动创建 VM
   └─ 调用 KVM 创建虚拟机
   └─ 自动配置 OpenClaw
   └─ 自动配置 Agent
   └─ 自动配置飞书渠道
   
7. 开始使用
   └─ 通过飞书与 Agent 交互
```

### 2.2 用户约束
- **每个用户只有一个 VM**
- **VM 规格随套餐固定**，用户不可调整
- **Token 配额**：根据套餐有不同配额

---

## 3. 套餐设计

### 3.1 套餐定义
**套餐 = 使用场景**（不是 VM 规格选择）

| 套餐 | 使用场景 | 预设 Agent | VM 规格 | Token 配额/月 |
|-----|---------|-----------|---------|--------------|
| 个人助理 | 日常事务 | 个人助手、日程管理、提醒 | 1核2G | 10万 |
| 数据分析 | 商业分析 | 数据分析师、报告生成 | 4核8G | 50万 |
| 财务分析 | 金融领域 | 财务分析、风险评估 | 4核8G | 50万 |
| 开发团队 | 软件开发 | Coder、Architect、Tester | 8核16G | 100万 |

### 3.2 套餐特点
- VM 规格随套餐固定，用户不可配置
- Agent 可调整（增删）
- 不同套餐对应不同的 qcow2 镜像

## 6. 计费方案

### 6.1 计费方式
**Token 计费 + VM 规格额外计费**

```
月费用 = 套餐基础费 + Token 超额费

套餐基础费 = VM 规格费用（按套餐固定）
Token 超额费 = (实际 Token - 配额) × 单价
```

### 6.2 Token 统计

**统计维度：**
- 按 Agent 统计
- 按时间统计（日/周/月）

## 9. 技术方案

### 9.1 虚拟化
- **KVM**
- 使用 qcow2 镜像
- 镜像由用户提供

### 9.2 自动化流程
```
用户确认
    ↓
调用 KVM API 创建 VM
    ↓
等待 VM 启动
    ↓
SSH 连接
    ↓
配置 OpenClaw
    ↓
配置 Agent
    ↓
配置飞书渠道
    ↓
启动服务
    ↓
通知用户
```

### 9.3 Token 统计
- OpenClaw Gateway 上报 Token 使用
- Redis 实时累加
- PostgreSQL 持久化
- 按时间、Agent、模型分组
