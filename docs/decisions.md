## 1. 用户注册方案

### 第一版（MVP）
- **注册方式**：邮箱 + 密码

## 新增决策（2026-03-04 23:19）

### 6. VM 创建流程
- **用户不主动创建 VM**
- 流程：选择套餐 → 选择 Agent → 自动创建 VM → 自动配置 OpenClaw → 可用
- VM 创建后自动完成：
  - OpenClaw 渠道配置
  - Agent 配置
  - 其他初始化

### 7. 配额限制
- **每个用户只有一个 VM**
- 配额主要是 **Token 配额**
- 无需限制 VM 数量

### 8. Token 统计展示
- 使用 **柱状图**
- 按 Agent 分组显示

### 9. 管理员后台（第一版需要）

**三大功能：**

1. **用户管理**
   - 用户列表/搜索
   - 查看用户详情
   - 禁用/启用用户
   - 查看用户的VM和消费

2. **Agent市场管理**
   - 添加/编辑/删除Agent模板
   - 配置Agent灵魂、人设、Prompt
   - 设置分类和标签
   - 启用/禁用Agent

3. **大模型管理**
   - 添加/编辑/删除模型配置
   - 设置定价（用于费用预估）
   - 启用/禁用模型
   - 测试模型连接

## 新增决策（2026-03-04 23:33）

### 11. 套餐定义
- **套餐 = 使用场景**（不是VM规格选择）
- 不同套餐包含不同的预设 Agent
- 示例：
  - 个人助理套餐：个人助手、日程管理、提醒
  - 数据分析套餐：数据分析师、报告生成
  - 财务分析套餐：财务分析、风险评估

### 12. VM规格
- **随套餐固定**，用户无法配置
- 不同套餐对应不同的VM规格
- 示例：
  - 个人助理套餐 → 1核2G
  - 数据分析套餐 → 4核8G
  - 财务分析套餐 → 4核8G

### 13. Agent可调整
- 选择套餐后，用户可以增删Agent
- 可以添加预设Agent
- 可以自定义创建Agent

### 14. 计费方式
- **Token计费** + **VM规格额外计费**
- Token：按实际消耗计费
- VM规格：按套餐对应的规格按时间计费

| 套餐 | 包含Agent | VM规格 | Token配额 |
|-----|----------|--------|----------|

| 数据分析 | 数据分析师、报告生成 | 4核8G | 50万/月 |
| 财务分析 | 财务分析、风险评估 | 4核8G | 50万/月 |
| 开发团队 | Coder、Architect、Tester | 8核16G | 100万/月 |

## 计费公式

```
月费用 = 套餐基础费 + Token超额费

套餐基础费 = VM规格费用（按套餐固定）
Token超额费 = (实际Token - 配额) × 单价
```
