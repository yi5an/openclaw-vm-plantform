## 快速开始

### 1. 注册账户

访问平台首页，点击"注册"按钮：

1. 填写邮箱地址
2. 设置密码（8-100字符）
3. 设置用户名（3-100字符）
4. 点击"注册"完成

**示例**:
```
邮箱: user@example.com
密码: MySecurePass123
用户名: myusername
```

### 2. 登录

使用注册的邮箱和密码登录：

1. 输入邮箱地址
2. 输入密码
3. 点击"登录"

### 3. 充值余额

在创建虚拟机之前，需要先充值：

1. 进入"我的账户"
2. 点击"充值"
3. 选择充值金额
4. 选择支付方式（支付宝/微信/银行卡）
5. 完成支付

### 4. 创建第一个虚拟机

1. 进入"虚拟机管理"
2. 点击"创建虚拟机"
3. 选择套餐（基础版/专业版/企业版）
4. 输入虚拟机名称（如：my-first-vm）
5. 点击"创建"

系统将自动：
- 分配资源（CPU、内存、磁盘）
- 创建虚拟机实例
- 安装 OpenClaw 环境
- 分配 IP 地址

创建过程约需 2-5 分钟，请耐心等待。

### 5. 启动虚拟机

创建完成后，虚拟机状态为"已停止"。点击"启动"按钮：

1. 状态变为"启动中"
2. 1-2 分钟后变为"运行中"
3. 显示分配的 IP 地址

### 6. 连接到虚拟机

**方式一：SSH 连接**

```bash
ssh root@<IP地址>
# 初始密码将在虚拟机创建后通过邮件发送
```

**方式二：Web 控制台**

1. 点击虚拟机详情
2. 点击"Web 控制台"
3. 在浏览器中直接操作

### 7. 配置 OpenClaw

虚拟机已预装 OpenClaw，登录后：

```bash
# 检查 OpenClaw 状态
openclaw status

# 启动 OpenClaw
openclaw start

# 配置 Telegram 渠道
openclaw config telegram

# 配置 WhatsApp 渠道
openclaw config whatsapp
```

## 虚拟机管理

### 虚拟机列表

进入"虚拟机管理"查看所有虚拟机：

显示信息：
- 名称
- 状态
- IP 地址
- 套餐
- 到期时间
- 操作按钮

**状态说明**:

| 状态 | 说明 | 可用操作 |
|------|------|---------|
| 创建中 | 正在创建虚拟机 | 无 |
| 运行中 | 虚拟机正常运行 | 停止、重启、SSH |
| 已停止 | 虚拟机已关闭 | 启动、删除 |
| 启动中 | 正在启动虚拟机 | 无 |
| 停止中 | 正在停止虚拟机 | 无 |
| 错误 | 虚拟机异常 | 重启、删除 |

### 创建虚拟机

#### 步骤1：选择套餐

根据需求选择合适套餐：

| 套餐 | CPU | 内存 | 磁盘 | 价格 | 适用场景 |
|------|-----|------|------|------|---------|
| 基础版 | 1核 | 2GB | 20GB | ¥99/月 | 个人使用、测试 |
| 专业版 | 2核 | 4GB | 50GB | ¥299/月 | 团队使用、生产环境 |
| 企业版 | 4核 | 8GB | 100GB | ¥599/月 | 高负载、多渠道 |

#### 步骤2：填写信息

- **名称**: 3-100字符，支持中英文、数字、下划线
- **区域**: 选择部署区域（默认：自动选择）
- **Agent模板**: 可选，选择预配置的模板

#### 步骤3：确认创建

- 显示费用明细
- 确认余额充足
- 点击"创建"

**注意事项**:
- 创建需要 2-5 分钟
- 扣除首月费用
- 无法重复创建同名虚拟机

### 启动虚拟机

1. 在虚拟机列表找到目标虚拟机
2. 确认状态为"已停止"
3. 点击"启动"按钮
4. 等待状态变为"运行中"（约1-2分钟）

**注意**:
- 首次启动可能较慢
- 请勿在启动过程中刷新页面

### 停止虚拟机

1. 在虚拟机列表找到目标虚拟机
2. 确认状态为"运行中"
3. 点击"停止"按钮
4. 等待状态变为"已停止"（约30秒-1分钟）

**注意**:
- 停止不会丢失数据
- 已停止的虚拟机仍会计费（直到到期）
- 请确保已保存重要数据

### 重启虚拟机

1. 在虚拟机列表找到目标虚拟机
2. 点击"更多" → "重启"
3. 等待虚拟机重新启动（约1-2分钟）

### 删除虚拟机

⚠️ **危险操作** - 删除后无法恢复！

1. 在虚拟机列表找到目标虚拟机
2. 确认状态为"已停止"
3. 点击"更多" → "删除"
4. 输入虚拟机名称确认
5. 点击"确认删除"

**删除前请确保**:
- 已备份重要数据
- 已导出配置文件
- 确认不再需要

### 续费虚拟机

#### 查看到期时间

虚拟机列表显示"到期时间"，提前7天会发送邮件提醒。

#### 续费步骤

1. 点击虚拟机详情
2. 点击"续费"
3. 选择续费月数（1-12个月）
4. 确认费用
5. 点击"确认续费"

**费用计算**:
```
续费金额 = 套餐月费 × 续费月数
```

**示例**:
- 套餐：基础版（¥99/月）
- 续费：3个月
- 费用：¥99 × 3 = ¥297

**Windows 用户**:

**Mac/Linux 用户**:

### 升级/降级套餐

#### 升级套餐

1. 进入虚拟机详情
2. 点击"升级套餐"
3. 选择新套餐
4. 补差价（按剩余天数计算）
5. 确认升级

**差价计算**:
```
差价 = (新套餐月费 - 旧套餐月费) × 剩余天数 / 30
```

**示例**:
- 当前：基础版（¥99/月）
- 升级：专业版（¥299/月）
- 剩余：15天
- 差价：(299 - 99) × 15 / 30 = ¥100

#### 降级套餐

⚠️ 降级操作限制较多，建议联系客服处理。

## 计费与充值

### 计费规则

#### 包月计费

- **计费周期**: 按月计费（30天）
- **扣费时间**: 创建虚拟机时扣除首月费用
- **续费时间**: 到期前或到期后续费

#### 费用组成

1. **基础费用**: 套餐月费
2. **额外费用**（可选）:
   - 额外磁盘: ¥0.5/GB/月
   - 自动备份: ¥20/月
   - 额外IP: ¥10/月

#### 余额不足

- 虚拟机到期时余额不足
- 系统将停止虚拟机
- 7天内续费可恢复
- 超过7天将删除虚拟机

## 常见问题

### 账户相关

#### Q: 忘记密码怎么办？

**A**: 
1. 点击登录页面的"忘记密码"
2. 输入注册邮箱
3. 查收重置邮件
4. 点击链接设置新密码

#### Q: 如何修改邮箱？

**A**: 
目前不支持自助修改邮箱，请联系客服处理。

#### Q: 账户被锁定怎么办？

**A**: 
多次登录失败会导致账户临时锁定（30分钟），请等待后重试或联系客服。

### 虚拟机相关

#### Q: 虚拟机创建需要多久？

**A**: 
通常 2-5 分钟。如超过 10 分钟仍为"创建中"，请联系客服。

#### Q: 虚拟机无法启动怎么办？

**A**: 
1. 检查账户余额是否充足
2. 查看虚拟机是否到期
3. 查看系统公告（可能维护中）
4. 联系客服处理

#### Q: SSH 连接失败怎么办？

**A**: 
1. 确认虚拟机状态为"运行中"
2. 检查 IP 地址是否正确
3. 检查密码是否正确（区分大小写）
4. 尝试使用 Web 控制台

#### Q: 虚拟机到期后数据会丢失吗？

**A**: 
- 到期后7天内续费，数据保留
- 超过7天未续费，虚拟机将被删除
- 建议提前备份重要数据

#### Q: 如何备份虚拟机数据？

**A**: 
1. **手动备份**: SSH 登录后使用 `tar` 打包
   ```bash
   tar -czf backup.tar.gz /path/to/data
   ```

2. **自动备份**: 开通自动备份服务（¥20/月）

3. **下载到本地**: 使用 SCP 或 SFTP 工具
   ```bash
   scp root@<IP>:/path/to/file ./
   ```

#### Q: 如何查看虚拟机日志？

**A**: 
1. SSH 登录虚拟机
2. 查看系统日志: `journalctl -u openclaw`
3. 查看应用日志: `tail -f /var/log/openclaw/app.log`

### OpenClaw 相关

#### Q: 如何配置 Telegram 渠道？

**A**: 
1. SSH 登录虚拟机
2. 运行 `openclaw config telegram`
3. 输入 Bot Token（从 @BotFather 获取）
4. 输入管理员用户ID
5. 运行 `openclaw restart`

#### Q: 如何配置 WhatsApp 渠道？

**A**: 
1. SSH 登录虚拟机
2. 运行 `openclaw config whatsapp`
3. 扫描二维码
4. 运行 `openclaw restart`

#### Q: 如何更新 OpenClaw？

**A**: 
```bash
# SSH 登录虚拟机
ssh root@<IP>

# 检查更新
openclaw update --check

# 执行更新
openclaw update

# 重启服务
openclaw restart
```

#### Q: OpenClaw 占用资源过高怎么办？

**A**: 
1. 检查运行的 Agent 数量
2. 减少不必要的渠道
3. 优化 Agent 配置
4. 考虑升级套餐

### 安全相关

#### Q: 如何提高虚拟机安全性？

**A**: 
1. **修改默认密码**: 首次登录后立即修改
2. **配置防火墙**: 只开放必要端口
   ```bash
   ufw allow 22/tcp
   ufw enable
   ```
3. **定期更新系统**:
   ```bash
   apt update && apt upgrade
   ```
4. **启用 SSH 密钥登录**（推荐）

#### Q: SSH 密码被修改后如何找回？

**A**: 
1. 使用 Web 控制台登录
2. 使用 `passwd` 重置密码
3. 如无法解决，联系客服重置

#### Q: 数据安全如何保障？

**A**: 
- 磁盘数据加密存储
- 网络传输使用 SSL/TLS
- 定期快照备份
- 多副本冗余存储
